Login

External Login

/account/external-login

Parameters

Name Required Type Desc
tenant Yes string
token Yes string JWT token
  • JWT token payload
    • brandCode
    • shopCode
  • Secret
    • YOUR_TENANT_SCRET

Example

Request:

pp://staging/account/external-login?tenant=YOUR_TENANT_NAME&token=JWT_TOKEN

Success:

{
  "result": {
    "token": "YOUR_TOKEN",
    "userId": 1000001,
    "userName": "saleswoman",
    "tenantId": 10,
    "tenantCode": "LABS",
    "tenantName": "LABS",
    "spots": [
      {
        "id": 2000001,
        "tenantId": 0,
        "name": "Event Hall",
        "colleagues": [
          {
            "userId": 1,
            "username": "700004567",
            "displayName": "将在秀"
          }
        ]
      },
      {
        "id": 2000002,
        "tenantId": 0,
        "name": "天山店",
        "colleagues": [
          {
            "userId": 1,

Create Token

/account/create-external-token

Parameters

Name Required Type Desc
secret Yes string
[KEY=VALUE] Yes string JWT payload

Example

Request:

pp://staging/account/create-external-token?secret=YOUR_SECRET&shopCode=C24A&userName=jackson

Success:

{
  "token": "NEW_TOKEN"
}
